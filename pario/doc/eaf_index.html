EXCLUSIVE ACCESS FILES

Basic Capabilities:

          Single portable interface to all filesystems

          Access to parallel filesystems

          Asynchronous read and write operations

          Persistent and scratch files

          Similar API to standard Unix file operations

          Built-in Pablo instrumentation

Available Architectures:

  • SunOS
  • AIX, workstations and SP-x configurations
  • SGI PowerChallenge
  • Intel Paragon
  • Cray T3D

Available file systems:

  • Unix File System (UFS)
  • Intel PFS
  • IBM PIOFS
